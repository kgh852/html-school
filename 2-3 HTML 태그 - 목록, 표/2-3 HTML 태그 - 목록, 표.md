# HTML 태그 - 리스트, 목록, 표

## 리스트, 목록
- `ol`: 순서가 있는 리스트
- `ul`: 순서가 없는 리스트
- `li`: 리스트의 내용을 지정하는 태그
- `dl`: 설명 목록을 만드는 태그
- `dt`: 설명 목록에서 이름을 지정하는 태그
- `dd`: 설명 목록에서 값을 지정하는 태그

---

## 표
### `table`: 표를 구현하는 태그
- `tr`: 행을 지정하는 태그
- `th`: 제목 행 내용을 지정하는 태그
- `td`: 행 안에 들어가는 내용을 정의하는 태그

---

### colspan, rowspan 속성
- `colspan`: 셀을 가로로 병합할 때 사용
- `rowspan`: 셀을 세로로 병합할 때 사용

---

### 열 지정 태그
- `col`: 열을 1개만 지정하는 태그
- `colgroup`: `col` 태그를 2개 이상 묶어서 여러 열을 지정하는 태그
